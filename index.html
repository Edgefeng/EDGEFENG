// PLANET
星球 · 瓦尔空伊
拥有两颗卫星的星球。一号卫星内斯尔疑藏有改变月都历史的秘密；二号卫星杜莱登表面为液态海洋，海底藏有科研设备无法探测的古文明遗迹。
// SPECIES · A
人类
称霸星球的生物，凭借超高智商与团队协作能力主宰文明。对法术的恐惧与觊觎，驱动着整个世界的历史走向。
// SPECIES · B
亚人
由人类创造的实验生物，具有动物特征。体格比人类更强壮，寿命更长，但普遍存在本能兽性冲动与智力局限。同样可在极端情况下觉醒成为术零，但概率更低。
// SPECIES · C
术零
觉醒法术的人类之统称。"法术的存在为零——需要立刻消灭"。平均每十年诞生一位，在强烈的极端情绪下觉醒。其出现伴随着光芒绕身，寿命得到延长。少数在极端情况下会引发毁灭级的"奇迹"现象。

奇迹 · 灰色国度
隆卡索国术零被杀害时爆发的法术，永久改变了周边数千公里环境。天空永不放晴，灰色雨水接触任何生物即引发病变腐烂，但接触无机物会立刻变为普通雨水。此地成为无人敢踏足之禁区，亦是国家灭亡之根源。
2377
奇迹 · 未知裂缝
一位术零为躲避追杀创造出的裂缝，踏入后音讯全无。此后踏入的科研人员与探险者亦无一生还，生死不明。裂缝至今静静存在，夫卡音涅国凭此奇迹改写国运。
2656
奇迹 · 胧紫沉眠之茧
一位术零沉睡于巨大紫色囊泡中。攻击者会被反射的囊孢命中，身上生长出膜状物体，直至将人完整包裹，化为新的茧。庇斯国以此奇迹为核威慑。
2938
奇迹 · 永不停歇的分裂
术零死于对死亡的极度恐惧时，身躯化为胶状物质。感知到恐惧的生物会被爬上包裹，发出哀嚎，随后胶状物质分裂复制，继续寻找下一个恐惧者。穆尼洛索国因此灭亡，现被月都科技穹顶封锁。
3346
奇迹 · 空无
直面虚无后无法承受而自杀的术零所留。踏入领域的有智慧生物会逐渐对现实感到无意义，丧失感知欲望，最终沦为无灵魂的空壳。
3480
奇迹 · 铁幕
不知名术零所留，位于奇迹空无附近。踏入者将陷入对自身法术、信仰、道德的无穷自我拷问，唯有完成逻辑自洽才能安全退出，否则理智永恒死锁。仅无知婴儿或完全失去理智者可全身而退。
4040
奇迹 · 树
"源"部落术零耗尽生命创造的参天巨树，初衷是让世界回归自然。然而巨树反而破坏了生态，其金色果实可治百病，但食用者一年后会皮肤硬化、长出枝桠，最终变成人形树木，继续蔓延。

反奇迹 · 灿烂绽晶
源自一枚微小水晶种子，可以指数级速度同化整个星球。空气中充满接触即绽放为花朵状利刃的晶体尘，地平线上的璀璨晶刺将平等地刺穿大部分人类。这是某人为了建立术零与人类共存世界而准备的终极手段。
